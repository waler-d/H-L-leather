P# Linksome 模拟笔试 — 练习测试 01

仿 [Apple 官网](https://www.apple.com/) 风格的徕卡/哈苏相机配件产品宣传页。

## 开始做题

1. 用 VS Code 打开本文件夹
2. 阅读 **`试题要求.md`**
3. Live Server 打开 `index.html` 预览
4. 用 Copilot + AI 修改代码，**勿删除 AI 对话记录**
5. 完成后回到 Cursor 对话交卷批改

## 文件说明

| 文件 | 用途 |
|------|------|
| `试题要求.md` | 主线任务 + 额外任务 |
| `评分标准.md` | 批改依据（100 分制） |
| `index.html` / `css/` / `js/` | 未完成起点代码 |
| `assets/` | 产品图与 hero.mp4 |
| `bak/` | 改崩可恢复 |

完整素材：`../图片、视频材料/`

## 静态部署说明

本项目为纯前端静态网站，不依赖 Node.js、Vue、React 或打包工具。

可直接将以下内容保持原目录结构上传到 GitHub Pages、Netlify 或 Vercel 等静态托管平台：

- `index.html`
- `css/`
- `js/`
- `assets/`

详细发布步骤见 `DEPLOY.md`。
